Preguntas

¿Cuál es la diferencia entre Internet y la World Wide Web?

Internet es la infraestructura global de redes que permite la comunicación. La World Wide Web (WWW) es un servicio que opera sobre Internet y permite acceder a documentos y recursos interconectados mediante hipervínculos.

¿Cuáles son las partes de un URL?

Las partes principales de un URL son:

  • Protocolo (ej. HTTP, HTTPS)
  • Dominio o dirección IP
  • Puerto (opcional)
  • Ruta del recurso
  • Parámetros de consulta (query parameters)
  • Fragmento o ancla (opcional)

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

  • GET: Solicita datos sin modificar el estado del servidor.
  • HEAD: Igual que GET, pero solo devuelve los encabezados.
  • POST: Envía datos al servidor para crear un recurso.
  • PUT: Actualiza un recurso existente o lo crea si no existe.
  • PATCH: Modifica parcialmente un recurso.
  • DELETE: Elimina un recurso.

¿Qué método HTTP se debe utilizar al enviar un formulario HTML?

El método POST se debe utilizar porque permite enviar datos de forma segura y no expone la información en la URL.

¿Qué método HTTP se utiliza al acceder a una página web?

El método GET es el que se utiliza al ingresar un URL en el navegador.

Un servidor web devuelve un código HTTP 200. ¿Qué significa?

Significa que la solicitud fue exitosa y no ocurrió ningún error.

¿Es responsabilidad del desarrollador corregir un error 404?

No siempre. Un error 404 significa que el recurso no se encontró. Puede deberse a un enlace roto o a que el usuario ingresó mal la URL.

¿Es responsabilidad del desarrollador corregir un error 500?

Sí, porque un error 500 indica un problema interno del servidor que debe ser corregido por el desarrollador.

¿Qué significa que un atributo HTML5 esté depreciado?

Significa que se desaconseja su uso y podría ser eliminado en futuras versiones. Ejemplos de elementos desaprobados de HTML 4 en HTML5 incluyen <font> y <center>.

¿Cuáles son las diferencias principales entre HTML 4 y HTML5?

  • HTML5 permite el uso de etiquetas semánticas como <article>, <section>, <header>, etc.
  • Soporta reproducción de audio y video sin plugins.
  • Mejor compatibilidad con dispositivos móviles.

¿Qué componentes de estructura y estilo tiene una tabla?

  • <table>: Contenedor principal.
  • <tr>: Filas.
  • <th> y <td>: Encabezados y celdas.
  • <caption>: Título de la tabla.

¿Cuáles son los principales controles de una forma HTML5?

  • Campos de texto (<input type="text">)
  • Botones de envío (<input type="submit">)
  • Controles numéricos (<input type="number">)
  • Controles de fecha (<input type="date">)
